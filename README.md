Run the server
==============

    git clone git@github.com:bmatsuo/mtrack.git $GOPATH/github.com/bmatsuo/mtrack
    cd $GOPATH/github.com/bmatsuo/mtrack
    make start-dist

then open [the web app](http://localhost:7890)
